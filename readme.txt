This plugin calls diff.io when posts are published or revised.

== Description ==

This plugin calls diff.io on every publishing or saving of a post.

== Installation ==

1. Install from the WordPress Store or upload the entire `wp-diffio` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access to the plugin settings and configure the key and secret.
